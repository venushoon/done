# 🎯 다했어요! 온라인 보드 (학생 주도형 과제 관리 시스템)

## 📌 시스템 개요
학생들이 스스로 부여된 과제를 수행하고, 자신의 이름표를 직접 이동시켜 완료를 인증하는 **자기 주도형 칸반(Kanban) 보드**입니다. 게임 요소(포인트, 폭죽 효과)를 결합하여 학생들의 성취 동기를 자극하고, 교사에게는 실시간 통계 및 누적 기록을 제공하여 학급 운영과 생활기록부 작성의 편의성을 극대화합니다.

## ✨ 주요 기능
* **직관적인 드래그 앤 드롭 UI**: 전자칠판 및 태블릿 환경에 최적화된 부드러운 애니메이션과 자유 배치(플로팅) 기능
* **스마트 명단 독바(Dock)**: 화면 위치(가로/세로)에 따라 최적화된 형태로 자동/수동 정렬되며, 화면을 가리지 않는 UI
* **즉각적인 시각/청각 피드백**: 과제 완료 시 경쾌한 효과음과 함께 폭죽 애니메이션(Confetti) 재생
* **영구 보존 포인트 시스템**: 과제별 난이도(1~3점)에 따른 포인트 자동 합산. 과제 목록을 삭제하거나 이름을 약간 다르게 적어도(띄어쓰기 등) 자동으로 통합되어 누적 장부에 영구 보존됨
* **플로팅 타이머**: 집중력 있는 과제 수행을 위한 화면 이동형 타이머 내장
* **교사 관리자 모드 (비밀번호 보호)**: 과제 및 명단 관리, 학생별 누적 데이터 확인, 수동 포인트 증감 등 강력한 중앙 통제 기능

## 👩‍🏫 교사 사용 설명서 (관리자용)
1. **관리자 모드 진입**: 우측 상단의 `🔒 교사 관리자` 버튼을 누르고 비밀번호(기본: 0000)를 입력합니다.
2. **과제 및 명단 세팅**: 
   * `📝 설정 및 관리` 탭에서 새로운 과제명, 이모지, 획득 포인트를 설정하여 추가합니다. (최대 6개)
   * 학생 이름을 입력하여 학급 명단을 구성합니다.
   * '학생 보드에 포인트 표시하기' 스위치를 켜고 꺼서 반 분위기에 맞게 경쟁 요소를 조절할 수 있습니다.
3. **현황 파악 및 누적 데이터 활용**:
   * `🏆 학생별 포인트 현황` 탭에서 아코디언 카드를 열어 학생별로 어떤 과제를 몇 회 수행했는지 영구 누적 기록을 확인합니다.
   * 필요시 포인트 수동 조절 버튼(`+`, `-`)을 활용하여 태도 점수 등을 부여합니다.

## 🙋‍♂️ 학생 사용 설명서
1. 선생님이 제시한 과제(예: 아침 독서, 우유 마시기 등)를 확인하고 스스로 수행합니다.
2. 하단(또는 측면)의 **'명단' 독바**에서 자신의 이모지 이름표를 찾습니다.
3. 자신의 이름표를 손가락으로 누른 채, 완료한 과제 칸으로 **드래그 앤 드롭** 합니다.
4. 폭죽이 터지고 내 포인트가 올라가는 것을 확인하며 성취감을 느낍니다! (잘못 넣었을 경우 이름표를 클릭하여 취소할 수 있습니다.)
